このスクリプトは、デスクトップの「ナレッジ変換」フォルダ内にある Office ファイル（Excel/PowerPoint）を PDF に自動変換し、動画・音声ファイル（mp4/mp3等）を AI で文字起こししてテキスト化するツールです。

## 1. 必要な環境と準備

本スクリプトを実行するには、事前に以下のソフトウェアとライブラリが必要です。

### ソフトウェア

LibreOffice: Office ファイルの PDF 変換に必須です。
[公式サイト](https://www.libreoffice.org/)からインストールしてください。
スクリプト内の `LIBREOFFICE_PATH` がインストール先と合致しているか確認してください（通常はデフォルトで動作します）。

### Python ライブラリ

コマンドプロンプトで以下のコマンドを実行し、必要なライブラリをインストールしてください。

```bash
pip install moviepy openai-whisper torch

```

※ `torch` (PyTorch) のインストールに時間がかかる場合があります。
※ `ffmpeg` がシステム環境変数に通っている必要があります（エラーが出る場合は `ffmpeg` をインストールし、パスを通してください）。

## 2. フォルダの準備

デスクトップに以下のフォルダ構造を作成してください。

1. デスクトップに 「ナレッジ変換」という名前のフォルダを作成します。
2. 変換したい Office ファイルや動画ファイルをこのフォルダに入れてください。
3. スクリプトを実行すると、処理済みの元ファイルは自動的に `C:\Users\ユーザー名\Desktop\ナレッジ変換\trash_backup` へ退避されます。

## 3. 実行方法

コマンドプロンプトでスクリプトを実行します。

```bash
python clean_local_folders.py

```

Office ファイル: 自動的に同じフォルダ内に PDF が生成されます。
動画/音声ファイル: 同じフォルダ内に `.txt` ファイルとして文字起こし結果が保存されます。
退避: 変換に成功した元のファイルは自動で `trash_backup` フォルダへ移動するため、元のファイルが混在して混乱するのを防ぎます。

## 4. 仕様と注意点

一時ファイル: 「~$」で始まるファイル（Office が編集中のファイル）はスキップする安全設計になっています。
エラーハンドリング: 変換や文字起こしに失敗した場合、元ファイルは移動されず、その場に残るようになっています。
Whisper AI: `model = whisper.load_model("base")` を使用しています。より高精度な変換が必要な場合は、`"small"` や `"medium"` に変更可能です。

## 5. 運用のヒント

このスクリプトを先ほど作成した `milaupload.py` と組み合わせることで、「変換・文字起こし」→「S3へ同期」という完全自動パイプラインが完成します。

1. ステップ1: `clean_local_folders.py` を実行して、手持ちの資料をすべて AI が読みやすい形式（PDF・テキスト）に統一する。
2. ステップ2: `milaupload.py` を実行して、変換後のデータを AWS S3 へクラウド同期する。

この流れをバッチファイル (`.bat`) にまとめておくと、デスクトップのアイコンをダブルクリックするだけで一括処理ができるようになります。
